## Installation

TabBarPicker is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "TabBarPicker" , :git => 'https://github.com/giuseppenucifora/TabBarPicker.git'

OR

pod "TabBarPicker"
```
